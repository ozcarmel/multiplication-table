# Learning Model

## Audience

The app serves grades 3-8:

- Grade 3: first structure and friendly facts.
- Grade 4: expanding fluency.
- Grade 5: harder facts and mixed practice.
- Grades 6-8: confidence repair, speed through understanding, and retrieval practice.

## Learning Order

Suggested order:

1. Identity facts: 1 times any number.
2. Doubles: 2 times.
3. Friendly anchors: 5 and 10.
4. Build from anchors: 3 and 4.
5. Bridge facts: 6.
6. Harder facts: 7, 8, and 9.
7. Mixed retrieval only after the child has strategies.

## Representation Stack

Each multiplication fact should be explainable in several ways:

- Table lookup: row by column.
- Array: groups arranged visually.
- Equal groups: the same group size shown several times, for example 4 groups of 3.
- Commutative fact: `a × b` equals `b × a`.
- Pattern: clue based on the factor.

## Practice Loop

The preferred loop is:

1. Explore a table cell.
2. Notice a pattern.
3. Answer one quick question.
4. Receive calm feedback.
5. Continue or stop while confidence is still high.

Short, repeated sessions beat long sessions.
