# Multiplication Table

Hebrew-first, bilingual multiplication learning web app for grades 3-8.

The first screen is a live 10x10 multiplication table. Kids can tap cells to see rows, columns, mirror facts, repeated addition, and dot arrays. The app also includes quick practice, a multiples hunt, confidence badges, printable charts, and parent/teacher guidance.

## Run Locally

```bash
npm install
npm run dev
```

## Product Principles

- Default Hebrew with full RTL layout; English is available with LTR layout.
- Start with patterns and confidence, not memorization pressure.
- Keep practice short, visual, and forgiving.
- Use rewards for exploration and persistence rather than speed alone.
- Support grades 3-8 through adjustable practice tracks.
